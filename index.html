iCar

iCar is a data-driven iOS application developed to change old model cars to smart
cars using IoT devices. The IoT sensors like GPS, Ultrasonic, Light, Temperature
sensors and RF Reader are used to connect to Raspberry Pi for collecting crucial
information and sharing it real-time to cloud. This allows user to monitor the car’s
information like location, speed, notifications through the phone.
‍
Technologies -
Swift, Python
UIKit, Core Location, MapKit, JSON, RESTful, User Notifications,  AVFoundation,
LocalAuthentication and Lottie.
